# Комплект ТЗ: Telegram-бот с мини-приложением (Tezo, розыгрыш, магазин)

Полноценный комплект технической документации для разработки проекта.  
Дата: 17.02.2025. Язык: русский. Формат дат: dd.mm.yyyy.

---

## Состав документов

| Документ | Назначение |
|---------|------------|
| **TZ.md** | Общее ТЗ: цели, Telegram-бот, мини-приложение, веб-админка, интеграции, логика розыгрыша |
| **TZ_Mini_App.md** | Подробное описание страниц мини-приложения (React): авторизация, розыгрыш, магазин, корзина, оформление |
| **TZ_Web_Admin.md** | Подробное описание страниц веб-админки: Магазин (каталог, карточка товара, S3), Розыгрыш, Заказы, Обращения, Пользователи и роли |
| **TZ_Backend.md** | ТЗ бэкенда: API для Mini App и Web Admin, Telegram-бот, интеграции (Tezo, ЮKassa, S3), логирование, безопасность |
| **TZ_Frontend.md** | ТЗ фронтенда: стек (React, TypeScript, Vite), структура Mini App и Web Admin, маршруты, компоненты, интеграция с API |
| **DB_Schema.md** | Схема БД PostgreSQL: таблицы, поля, связи, индексы |

---

## Порядок изучения

1. **TZ.md** — общее понимание проекта.
2. **DB_Schema.md** — структура данных.
3. **TZ_Backend.md** — серверная часть и API.
4. **TZ_Frontend.md** — клиентская часть.
5. **TZ_Mini_App.md** и **TZ_Web_Admin.md** — детали экранов и страниц.
