# Mini App (Telegram Web App)

React-приложение: розыгрыш, магазин, корзина, оформление заказа.

**ТЗ:** см. `docs/TZ_Mini_App.md`, `docs/TZ_Frontend.md`

---

## Стек

React 18, TypeScript, Vite, React Router, React Query.  
Инициализация через Telegram Web App SDK (`initData`).

---

## Маршруты

`/`, `/auth`, `/raffle`, `/raffle/play`, `/shop`, `/cart`, `/checkout`.
