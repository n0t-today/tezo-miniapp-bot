# Web Admin

Веб-админка: товары, розыгрыши, заказы, обращения, пользователи и роли.

**ТЗ:** см. `docs/TZ_Web_Admin.md`, `docs/TZ_Frontend.md`

---

## Стек

React 18, TypeScript, Vite, React Router, React Query, UI-библиотека (Ant Design / MUI / shadcn).  
Авторизация: JWT (логин/пароль или Telegram).

---

## Маршруты

`/login`, `/admin`, `/admin/raffle`, `/admin/shop`, `/admin/orders`, `/admin/appeals`, `/admin/users`.
